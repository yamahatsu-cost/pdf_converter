PDF ⇔ 画像 変換

山發日本株式会社　社内アプリ

1. PDF → 画像

PDFを1ファイル読み込み、各ページを画像として出力します。保存名は 元のPDF名-1P.jpg または 元のPDF名-1P.png です。

PNG選択時はこの設定は使われません。
PDFを選択してください。

2. 画像 → PDF

JPG / JPEG / PNG を複数読み込み、最初にファイル名の数字順で整列します。 その後、ドラッグまたは上下ボタンでページ順を手動変更できます。

スマホでは上下ボタン、PCではドラッグでも並べ替えできます。
画像を1枚以上選択してください。

3. PDF文字おこし

PDFから文字情報を抽出して表示します。文字情報を持つPDFに対応しています。 スキャン画像だけのPDFでは、文字がほとんど取得できない場合があります。

PDFを選択してください。
外部翻訳に送る
文字抽出後に「翻訳準備」を押すと、翻訳先言語の設定とチャンク分割を行います。
分割前 選択なし 訳文 0件
翻訳チャンク一覧 まだ分割されていません
保存済み訳文の結合結果 まだありません
画面が見づらくならないよう、訳文入力欄は選択中チャンクだけ表示します。翻訳元言語は外部サービス側で自動判定します。機密情報の送信には注意してください。

4. PDF → Word

PDFから抽出したテキストを、Wordで開ける .doc 形式で保存します。 文字情報を持つPDFに対応しています。繰り返し出るヘッダー・フッター候補は自動除外します。

PDFを選択してください。

5. 画像OCR

スキャンPDFだけでなく、JPG / PNG 画像からもOCRで文字を抽出します。 処理はブラウザ内で行われますが、ページ数や画像枚数が多い場合は時間がかかることがあります。

PDFまたは画像を選択してください。
OCR認識画像プレビュー
OCR前に白黒強調した画像を表示します。PDFをOCRした場合も各ページの認識画像を確認できます。

5. 使用方法

・PDF→画像 は複数ページをまとめて ZIP で保存します。
・画像→PDF は最初に数字順、数字がない場合は名前順に並びます。
・ブラウザ上で処理するため、ファイルはサーバーへ送信しません。
・PDF文字おこしは、文字情報を含むPDFからテキストを取り出します。
・GitHubに上書き更新したときは、アプリ内バージョンを変更すると更新案内を表示できます。
